# FKApplyDesign
Run
------
To Run <br>
git clone https://github.com/manansingla4/FKApplyDesign.git <br>
cd FKApplyDesign <br>
git checkout Dev <br>
cd src <br>
javac *.java <br>
java Main <br>
In your terminal <br>
Enter Coordinates of the box and top left corner being 1 1 (Basic Coordinate Convention for Matrix)
